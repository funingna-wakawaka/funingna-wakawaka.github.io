关于Github的高校认证

software

关于github的高校认证(很不幸,由于有太多根本不是学生的人薅羊毛,github在4月20日取消了学生认证,但已认证者仍可享受以下权益)

​

这里我就说直观的好处吧

Github Copilot Pro(默认自带)

这个就是随便使用各种模型,除了claude要钱

​

DigitalOcean(必要)

这个可以开服务器,会赠送学生200美元的额度

​

​

Termius(必要)

这个会赠送学生专业版本,这个是负责管理服务器的

​

namecheap和Name.com(不推荐,除非你可以承担的续订域名的费用)

这个会赠送学生域名和SSL证书什么的(但我十分不推荐,因为真的好贵呀,一年过后续订至少45美元,还不如阿里云一年不到50块的域名)

​

​

​

Azure(必要)

这个会给学生100美元的额度

​

Jetbrains(必要)

这个就不用说了吧,大名鼎鼎的IDEA java编辑器就是他们的,这个会给学生免费使用他们的工具和AI

lambdatest(可选)

这个是一个在线跨浏览器测试平台,我们可以直接在平台上使用提供的设备测试自己的网站对对应设备(比如苹果手机)的兼容性

不过只有一年免费时间

​

Imgbot(必要)

这个可以安装在github来优化blog的图片来提高blog的访问速度,学生免费哦
