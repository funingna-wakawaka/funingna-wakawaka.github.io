win加R命令和Windows快捷键

diary

以下是一些常用的命令和快捷键,用Chatgpt统计了一下

Win+R 命令

一、系统工具类

  • cmd → 命令提示符
  • powershell → PowerShell
  • taskmgr → 任务管理器
  • msconfig → 系统配置（启动项管理）
  • services.msc → 服务管理
  • eventvwr → 事件查看器
  • perfmon → 性能监视器

二、设置与控制面板

  • control → 控制面板
  • settings → Windows 设置
  • appwiz.cpl → 程序和功能（卸载软件）
  • optionalfeatures → Windows 功能开关
  • sysdm.cpl → 系统属性
  • desk.cpl → 显示设置（旧版）

三、磁盘与设备

  • diskmgmt.msc → 磁盘管理
  • devmgmt.msc → 设备管理器
  • cleanmgr → 磁盘清理
  • dfrgui → 磁盘碎片整理

四、网络相关

  • ncpa.cpl → 网络连接
  • inetcpl.cpl → Internet 选项
  • ipconfig（需 cmd）→ 查看IP
  • firewall.cpl → 防火墙

五、文件与路径

  • explorer → 文件资源管理器
  • shell:startup → 启动文件夹
  • shell:desktop → 桌面
  • shell:downloads → 下载目录
  • %temp% → 临时文件夹

六、安全与用户

  • netplwiz → 用户账户管理
  • secpol.msc → 本地安全策略（部分版本没有）
  • lusrmgr.msc → 本地用户和组（专业版以上）

七、高级/开发常用

  • regedit → 注册表编辑器
  • gpedit.msc → 组策略（专业版）
  • dxdiag → DirectX诊断工具
  • verifier → 驱动验证工具

八、一些隐藏但超实用

  • winver → 查看Windows版本
  • mrt → 恶意软件删除工具
  • osk → 屏幕键盘
  • magnify → 放大镜
  • snippingtool → 截图工具

  • Tab 自动补全路径

  • Ctrl + C 强制停止命令
  • 右键可以粘贴

Windows 快捷键

一、基础操作

  • Ctrl + C：复制
  • Ctrl + V：粘贴
  • Ctrl + X：剪切
  • Ctrl + Z：撤销
  • Ctrl + Y：重做
  • Ctrl + A：全选

二、文件 / 资源管理器

  • Win + E：打开文件资源管理器
  • Alt + ↑：返回上一级目录
  • Ctrl + Shift + N：新建文件夹
  • F2：重命名文件
  • Delete：删除
  • Shift + Delete：彻底删除（不进回收站）

三、窗口管理（超级常用）

  • Win + D：显示桌面 / 恢复窗口
  • Win + L：锁屏
  • Alt + Tab：切换窗口
  • Alt + F4：关闭当前窗口
  • Win + ↑ / ↓ / ← / →：窗口最大化 / 最小化 / 分屏
  • Win + Tab：任务视图

四、浏览器通用快捷键（几乎所有浏览器都一样）

  • Ctrl + T：新建标签页
  • Ctrl + W：关闭当前标签
  • Ctrl + Shift + T：恢复关闭的标签
  • Ctrl + Tab：切换标签
  • Ctrl + L：选中地址栏
  • Ctrl + R / F5：刷新

五、搜索 & 系统

  • Win + S：打开搜索
  • Win + R：运行（输入命令）
  • Win + I：打开设置
  • Win + Pause：系统信息

六、高效办公（强烈推荐）

  • Ctrl + F：查找
  • Ctrl + H：替换
  • Ctrl + P：打印
  • Ctrl + S：保存
  • Ctrl + Shift + Esc：任务管理器

七、虚拟桌面（很多人不知道）

  • Win + Ctrl + D：新建桌面
  • Win + Ctrl + ← / →：切换桌面
  • Win + Ctrl + F4：关闭当前桌面

八、截图（非常实用）

  • Win + Shift + S：区域截图（推荐🔥）
  • PrtSc：全屏截图
  • Alt + PrtSc：当前窗口截图

九、进阶（高手常用）

  • Win + X：打开高级菜单（设备管理、终端等）
  • Win + .：打开表情面板 😊
  • Win + V：剪贴板历史（需开启）
  • Win + G：游戏栏
