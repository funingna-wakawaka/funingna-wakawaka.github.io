PowerToys 和 PSD Codec

software

最近发现电脑下卡通图片的时候发现电脑不能预览 svg图片,于是我发现了PowerToys

PowerToys

这个软件还挺火的,长得和微软的windows设置界面一模一样

点击下载发行版 Release v0.97.1 · microsoft/PowerToys · GitHub

​

​

我们一般情况下电脑不会显示出 svg 图片,这个可以支持svg和pdf

​

开启后可以发现这个svg可以预览了 此时一张svg混入png其中……..狗头狗头狗头

​

PSD Codec

点击下载PSD Codec - Download

这个是我在B站上看到的一个软件,up主给了破解方式,大家自用就行了

原版安装后进入下面目录
C:\Program Files\ArdfryImaging\PSD Codec
双击那个大写的A图标exe文件……输入下面激活码中任何一个激活码，搞定！

tam@core.com
MHXB-NCU3-HGYT-AC8J

2@2.com
G5BE-P5UR-J2QW-SRBQ

作者：殇の羽落

出处：bilibili

​

这个软件可以支持psd docx pptx 以及一大堆常见文件的缩略图

​

其实感觉用处不大,因为我找文件只看名字,但svg图片是非常有必要添加一个预览图的
